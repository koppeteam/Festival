# Yallapur Police - Jaatre Information Center

A bilingual (Kannada/English) React application for managing festival information.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add logo images to `public/assets/`:
   - logo1.jpg (Karnataka emblem)
   - logo2.png (Namma 112)

3. Start the development server:
```bash
npm start
```

## Features

- Bilingual support (Kannada & English)
- Responsive design
- YouTube video integration
- Navigation sections for:
  - Gallery
  - Traffic Information
  - Parking Information
  - Emergency Contacts
  - Public Toilets
  - Safety Precautions

## Building for Production

```bash
npm run build
```
